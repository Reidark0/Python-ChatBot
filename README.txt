# Atendimento Automatizado para Barbearia 90S

Este projeto é uma solução proposta pelo grupo Squad9, composto por entregadores da plataforma iFood, como parte do curso Vamoai patrocinado pela empresa de educação Resilia.

## Tecnologias utilizadas

- Python

## Como usar

1. Clone este repositório.
2. Execute `python main.py` para iniciar o servidor.
3. Acesse `http://localhost:5000` para acessar o sistema.

## Funcionalidades

- Agendamento de serviços
- FAQ
- Sobre nós
- Canais de atendimento
